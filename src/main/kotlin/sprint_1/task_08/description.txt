Научиться использовать Java для хранения и группировки объектов с помощью коллекций HashSet и HashMap.

### Требования:

1️⃣ Создайте класс **Product** с полями:

- **id** (уникальный идентификатор);

- **name** (имя продукта);

- **category** (категория продукта, тип Category).


2️⃣ Создайте **enum Category** с возможными категориями товаров (например, **FOOD, ELECTRONICS, CLOTHING, OTHER**).

3️⃣ Создайте класс **ProductManager**, который будет отвечать за управление товарами. В этом классе создайте:

- Поле **Set products**, хранящее все объекты продуктов.

- Поле **Map<Category, List> categoryMap**, где ключом выступает категория, а значением — список продуктов, принадлежащих этой категории.


4️⃣ Реализуйте следующие методы в **ProductManager**:

- **addProduct(Category category, String name)**: создает новый продукт с указанной категорией и именем, добавляет его в множество продуктов и обновляет categoryMap.

- **removeProduct(Category category, String name**): находит продукт с указанным именем в указанной категории, удаляет его из множества продуктов и из списка в categoryMap.

- **findProductsByCategory(Category category)**: возвращает список всех продуктов в указанной категории. Если категория отсутствует, метод возвращает пустой список.

- **groupProductsByCategory()**: обновляет categoryMap, группируя все текущие продукты в множество по их категориям.

- **printAllProducts()**: выводит список всех продуктов, сгруппированных по категориям, в формате:


```
Категория: FOOD
Продукты:
- Apple
- Bread

Категория: ELECTRONICS
Продукты:
- Laptop
- Smartphone
```

5️⃣ Реализуйте класс **Main** для тестирования работы системы. Создайте примеры добавления, удаления и поиска продуктов.
